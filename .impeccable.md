## Design Context

### Users
Tech-savvy professionals, recruiters, and potential clients visiting a developer portfolio. Multilingual audience (EN, NO, UA). They want to quickly assess skills, read tech news/blog, and find contact information. Mobile and desktop users.

### Brand Personality
Bold, Technical, Modern

### Aesthetic Direction
- **Visual tone**: Dark glassmorphism with vibrant neon accents per section. Premium, app-like feel.
- **Theme**: Dark-first UI for overlays/modals (#1A1730 base), light content areas
- **Font**: Comfortaa (rounded geometric, good Cyrillic) for brand/headers, system sans for body
- **Color system**: 6 distinct neon section colors (BentoGrid), indigo (#6366F1) as primary action, amber (#fbbf24) as secondary accent
- **Anti-references**: Generic corporate templates, flat/boring portfolios, cookie-cutter Bootstrap sites
- **References**: Interactive developer portfolios with personality, app-like navigation on mobile

### Design Principles
1. **Personality over polish** — Each section has its own color identity; the grid layout is distinctive, not generic
2. **Motion with purpose** — GSAP/Framer Motion animations reinforce interactions (hover reveals, particle effects), not decoration
3. **Mobile as app** — Bottom navigation, swipe gestures, glassmorphism — feels native, not a shrunk desktop
4. **Content density** — News/blog-heavy site needs efficient information hierarchy without clutter
5. **Accessibility baseline** — WCAG AA, 44px touch targets, reduced motion support, proper focus indicators

### Technical Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS 3.4 + tailwindcss-animate
- Framer Motion, GSAP, Three.js (particles)
- Radix UI primitives (dialog, tooltip, scroll-area)
- Lucide icons

### Spacing
- Base radius: 12px (0.75rem)
- Desktop grid gap: 20px
- Mobile grid gap: 12px
- Touch target minimum: 44x44px

### Typography
- Brand: Comfortaa 400/700, Latin + Cyrillic
- Body: System sans-serif stack
- Fluid sizing via clamp() for responsive headings
